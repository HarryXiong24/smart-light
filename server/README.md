# Smart Light API Doc

## Handle Light State


GET http://localhost:8080/handle-status

### Parameters

#### isOpen

0 - close light

1 - open light

### response

```json
{
  "execTime": "2024-11-22T20:27:02.340568-08:00",
  "isOpen": 0,
  "status": 0
}
```
